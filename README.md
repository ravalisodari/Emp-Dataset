# 📊 Employee Dataset Analysis

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on an Employee Dataset to uncover insights about employee demographics, salaries, departments, job roles, and performance. The analysis uses Python and popular data science libraries to clean, visualize, and interpret the data.

---

## 🎯 Objectives

- Clean and preprocess employee data.
- Analyze employee demographics.
- Compare salaries across departments.
- Explore age and experience distributions.
- Identify trends in employee performance.
- Create visualizations for better decision-making.

---

## 📂 Dataset

The dataset contains employee information such as:

- Employee ID
- Name
- Age
- Gender
- Department
- Job Role
- Salary
- Experience
- Education
- Performance Rating

> **Note:** The dataset is intended for educational and data analysis purposes.

## ▶️ Getting Started

### Clone the repository

```bash

```

### Install dependencies

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

### Run the notebook

```bash
jupyter notebook
```

Open `Employee_Analysis.ipynb` and run all cells.
